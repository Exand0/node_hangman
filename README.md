# Node Hangman
CLI Hangman game for a CS Course Informatics and Society at Vienna University

# About
This Hangman game uses terms from the history of computer science. These range from antiquity to the recent past, from people to companies and from technologies to products.

# How to play
Clone the source code and run with node. (version v13.8.0 was used for development)
